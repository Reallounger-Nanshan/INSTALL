## 一、硬件安装

### 1.1 组装主板

#### 1.1.1 安装固态硬盘

* 固态硬盘位易被塔式散热器和显卡挡住，故推荐优先安装

#### 1.1.2 安装 CPU

* 打开 CPU 扣具
  * **勿拆 CPU 扣具**，否则可能会导致拆卸散热器使 CPU 掉落或划歪主板上的针脚等问题
* 装上 CPU
  * Intel CPU 其中**一角**和主板对应位置有**三角形标识**以确认安装方向
* 按下卡扣
  * CPU 保护盖会自动弹出

#### 1.1.3 安装散热器

* 支持 AMD CPU 的主板有原装散热器扣具，需拆除
  * **支持 Intel CPU 的主板没有原装散热器扣具**——避免误拆 CPU 扣具
* **安装散热器扣具**
  * 具体方式需参照散热器说明书
* 在 CPU 上**涂抹适量硅脂**并用小铲子抹平
* **撕除**散热器底座**保护膜**
* **安装塔式散热器**
* 将**挂钩安装在散热器风扇上**
* **将散热器风扇安装在散热器上**
  * 风扇风向朝向内存条

* 将**风扇供电线插到主板上的 CPU_FAN处**

### 1.2 将主板安装至机箱

#### 1.2.1 安装主板挡板

* 封装较好的主板本身就固定在主板上，无需单独安装

#### 1.2.2 安装主板

* 拆卸主板各块挡板

* 对准主板和机箱的螺丝孔位，安装螺丝

### 1.3 安装内存条与显卡

#### 1.3.1 安装内存条

* 内存条卡槽有防呆设计，通过比对两段长度确定方向
* 内存插槽选择
  * 单内存条
    * 1 通道 (不推荐)
      * 容易挡住散热器
    * 2 通道
    * 3 通道
    * 4 通道
  * 双内存条
    * 1、3 通道 (不推荐)
    * 2、4 通道
  * 四内存条
    * 全部插满

* 取出内存时需要先按下卡扣

#### 1.3.2 安装显卡

* 拆卸机箱上的显卡挡板

* 风扇向下安装显卡
  * 安装好后，会听到一声清脆的卡扣关闭声音
  * ==**取出显卡时需要先按下卡扣**==
    * 用力会导致卡扣坏掉——不影响使用，但无法彻底固定显卡

### 1.4 连线

#### 1.4.1 电源连线

* 24-pin (20 + 4) 电源线连接主板
* 标注有 600W 的电源线
* 标注有 CPU 的 8-pin (4 + 4) 电源线连接主板上的 CPU 供电口
  * 有多少插多少

#### 1.4.2 机箱前面板连线

* USB3.0 (19-pin，一角缺失一个口用于防呆) 线插在主板上的 U32 接针上
* 9-pin (第一排第四个针脚缺失) 线插在主板上的前面板音频接针上
* 9-pin (第一排第五个针脚缺失) 线插在主板上的系统控制面板接针上

### 1.5 测试

#### 1.5.1 测试

* 连接电源
* 连接显示器
* 开机
* 如果主板指示灯按顺序逐一亮灭，显示器出现主板 LOGO 并正常进入 BIOS 则安装成功

#### 1.5.2 故障排除

* CPU 指示灯 (**红灯**) 常亮
  * 含义
    * CPU 自检未通过
  * 可能导致的问题
    * 主板接针歪斜 / 损坏
    * CPU 损坏
* DRAM 指示灯 (**黄 / 橙灯**) 常亮
  * 含义
    * 内存自检未通过
  * 可能导致的问题
    * 内存条接触不良
      * 未插紧——重新插拔
      * 金手指上存在异物——用干净纸巾擦拭
    * 内存条损坏
    * 主板的内存条插槽损坏
    * 连接内存条的主板接针歪斜 / 损坏
* VGA 指示灯 (**白灯**) 常亮
  * 含义
    * 显卡自检未通过
  * 可能导致的问题
    * 显卡接触不良
      * 未插紧——重新插拔
      * 金手指上存在异物——用干净纸巾擦拭
    * 显卡供电不稳定
      * 电源线未插紧——重新插拔
      * 电源线掉落——重新插入
        * 即使电源线未插入，只要金手指接触正常，显卡自带灯光依然会被点亮
    * 显卡损坏
    * 数据线损坏
* BOOT 指示灯 (**黄灯**) 常亮
  * 含义
    * 没有检测到 BOOT 启动项文件——新电脑属于正常情况

### 1.6 完成组装

* 安装机箱各挡板

#### 2.1.2 下载 ISO 文件

 ### 2.2 制作启动盘

#### 2.2.1 格式化 U 盘

* 右键格式化 U 盘——选择 NTFS 文件格式
  * 文件格式
    * 概念
      * 系统对文件的存放排列方式，不同格式的文件系统关系到数据是如何在磁盘进行存储，文件名、文件权限和其他属性也存在不同
    * 分类
      * FAT32
        * 特点
          * **通用**格式，任何USB存储设备都会预装该文件系统，可以在任何操作系统平台上使用
        * 缺点
          * **最大单文件大小容量只支持 4GB**
      * exFAT
        * 特点
          * Microsoft 创建的用来**取代 FAT32 文件格式**的新型文件格式
          * 最大可以支持 1EB 的文件大小，非常适合用来**存储大容量文件**
          * 可以在 Mac 和 Windows 操作系统上通用
        * 缺点
          * **没有文件日志功能**，不能记录磁盘上文件的修改记录
      * NTFS (New Technology File System)
      * 特点
        * Microsoft 为硬盘或固态硬盘 (Solid State Drive, SSD) 创建的默认新型文件系统
        * **集成了几乎所有文件系统的优点**：日志功能、无文件大小限制、支持文件压缩和长文件名、服务器文件管理权限等
      * 缺点
        * Mac 系统只能读取 NTFS 文件但没有权限写入，需要借助第三方工具才能实现——跨平台的功能较差

### 2.3 安装系统

#### 2.3.1 通过 U 盘启动盘安装系统

* 将启动盘插入电脑

#### 2.3.2 进入 BIOS

* 如果 BIOS 和 boot menu 集成在一起
  * 直接进入 boot menu 并选择 U 盘
* 如果 BIOS 和 boot menu 没有集成在一起
  * 进入 BIOS
  * 把 USB Boot 设置为 Enable
  * 关闭安全启动
  * 修改启动优先级——USB Boot 最高
  * 保存设置并退出
  * 安装好系统后需要将设置改回原设置

#### 2.3.3 安装系统

* 设置语言和其他首选项

* 点击“现在安装”

* 选择“仅安装 Windows"

  * 新电脑无其他系统选择另一项无效

* 选择 Windows 系统安装位置

* 设置地区

* 设置输入法

* 联网或跳过

  * 如果**无法正常联网且无法直接跳过**

    * Shift + F10 开启终端

    * ```
      oobe\bypassnro
      ```

    * 重启

    * 选择“我没有 Internet 连接”

* 设置账户名

### 2.4 缺少无线网卡驱动

#### 2.4.1 问题来源

* 华硕天选 B760M WIFI Ⅱ 主板

#### 2.4.2 解决方案

* 有线联网 / USB 共享网络
* 系统提示安装华硕主板的控制软件——Armoury Crate
* 安装完后重启即可发现无线网卡驱动



## 三、软件安装

### 3.1 网线直连传输文件

#### 3.1.1 网线连接两台电脑

* 将网线插入两台电脑网口
  * 网线性能影响传输速度

#### 3.1.2 设置固定 IP

* 网络和 Internet
* 高级网络设置
* 更多网络适配器选项
* 选择以太网，右键选择属性
  * 网络——Internet 协议版本 4 (TCP/IPv4)——属性
    * 设置 IP 地址和子网掩码——确定——确定
      * 电脑 1
        * IP 地址：192.168.0.1
        * 子网掩码：255.255.255.0
      * 电脑 2
        * IP 地址：192.168.0.2
        * 子网掩码：255.255.255.0
  * 共享——Internet 连接共享——允许其他网络用户通过此计算机的 Internet 连接来连接

#### 3.1.3 关闭防火墙

* Windows 安全中心——防火墙和网络保护——关闭公用网络防火墙

#### 3.1.4 共享文件

* 选择需要传输的文件夹
* 右键“属性”——共享——高级共享——共享此文件夹——权限——修改权限——确认——确认

#### 3.1.5 查看共享文件夹

* 在文件资源管理器中输入目标电脑的 IP 地址
* 填写目标电脑的账户密码
